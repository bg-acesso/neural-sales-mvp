**Briefing Criativo Detalhado**

**Título:** Otimização de Conteúdo para Engajamento

**Mood e Atmosfera:**
A imagem deve transmitir uma sensação de inovação, eficiência e engajamento. O estilo é corporate com elementos de tecnologia, visando criar uma atmosfera moderna e profissional. A emoção principal é a de otimismo e melhoria contínua.

**Elementos Visuais:**

* **Composição:** Minimalista com elementos geométricos, utilizando linhas limpas e espaços em branco para criar uma sensação de claridade e foco. A composição deve ser equilibrada, com um ponto focalprincipal que oriente o olhar do espectador.
* **Paleta de Cores:** Baseada na emoção do post, a paleta deve incluir cores como azul (#4567b7) para confiança e profissionalismo, verde (#8bc34a) para crescimento e otimismo, e cinza (#808080) para neutralidade e equilíbrio. Essas cores devem ser aplicadas em gradientes suaves para criar um efeito visual atraente.
* **Estilo Técnico:** 3D render com elementos de ilustração vetorial para criar uma imagem única e futurista. O uso de luz e sombra deve ser realista, destacando os elementos principais da composição.

**Referências de Design:**
As tendências de design 2025/2026 aplicáveis incluem o uso de interfaces minimalistas, tipografia ousada e a integração de elementos de realidade aumentada. No feed do LinkedIn, estilos que se destacam incluem imagens com temas de inovação tecnológica, sustentabilidade e desenvolvimento pessoal. A imagem deve incorporar esses elementos de forma criativa e relevante.

**Considerações Finais:**
A imagem final deve ser atraente, fácil de entender e transmitir a mensagem de otimização de conteúdo para engajamento de forma clara e convincente. O uso de texto deve ser minimalista, com foco em uma mensagem principal ou chamada para ação. A imagem deve ser escalável para diferentes formatos de publicação, mantendo sua integridade visual e impacto emocional.
